# Snapshot V3与显式MCP映射闭环

时间：2026-07-18 17:11 CST

## 裁决

- MCP Discovery采用`MCPCapabilitySnapshotV3`直接保存Page/Receipt/Apply/Material Set及逐项
  Tool/Resource/Prompt Material exact provenance，不另立Snapshot外部来源索引，也不改写V2历史。
- MCP Tool进入Praxis Registry必须提交版本化`MCPToolMappingManifestV1`；不按名称、Schema、
  annotations或模型推断自动映射。
- Package离线Verify与Admission复用OCI content-addressed Artifact、Sigstore Bundle和in-toto
  Statement；Verification、Package current、Trust Policy、Artifact exact必须在同一CAS复读。

  Admission完成；generic MCP Tool/Mapping Transition fail closed，V1只到admitted。
