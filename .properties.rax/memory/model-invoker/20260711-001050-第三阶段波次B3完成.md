State只保留下一次请求必须原样回传的 `thinking`、`redacted_thinking`和 direct `tool_use`块。普通 assistant text、未知内容块、非 direct caller、顶层或 caller `cache_control`和额外字段继续在 HTTP前拒绝，避免 State变成不受控原生输入通道。

直接测试使用 `acme-messages`非原厂 Binding完成 thinking → tool result → final text往返，证明 Response、State、MappingReport、流终态和错误身份均来自 Binding。SDK错误对象、HTTP request、Credential和任意 native cause不进入公开 unwrap链；只有 context安全 sentinel可以保留。
