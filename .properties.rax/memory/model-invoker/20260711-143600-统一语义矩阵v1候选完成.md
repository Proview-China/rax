## 关键决定

1. Native/Compatible/Partial/Unsupported分别映射exact/transformed/degraded/rejected；Partial仍需显式`AllowDegradation`。
2. Message.Text、FunctionResult.Output、function Tool、OutputItem与StreamEvent未来均可保留旧字段语义后加tag/可选字段，不需要现在破坏性修改。
3. Profile消费Tool/Capability/Mapping；Cache消费Route与Provider缓存事实；Runtime/Context消费Gateway和统一调用合同。CacheIntent仍待跨线程联合决定。
4. xAI gRPC、Qwen DashScope、多模态、Hosted Tools、Batch、Realtime、后台执行明确延期；Meta Llama精确Route和CacheIntent需用户/相邻线程决定。
