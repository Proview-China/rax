# 2026-07-17 Review 企业多签 V2 语义冻结

## 用户裁决

- production State Plane：先采用单机 SQLite，明确无 HA/SLA，公开 Port 保持可迁移；
- 路由、Rubric、quorum 与预算：必须来自租户版本化 current Policy；缺失即 Fail Closed，不使用代码猜测默认承诺；
- Human production：企业多签，不以现有单 Reviewer V1 冒充；
- quorum：租户 Policy 定义 K-of-N 与必需角色，按 distinct human identity 计票；
- veto：具备 current veto 权的 Reviewer Reject 为硬否决；
- delegation/self-review：只接受 Authority Owner current 显式 Delegation，production 禁止作者/责任主体自审。
