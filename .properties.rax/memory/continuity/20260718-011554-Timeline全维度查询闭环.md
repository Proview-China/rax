# Timeline全维度查询闭环

时间：2026-07-18 01:15 CST

## 结论

`tmp.document/Continuity.md`明确要求的Timeline查询维度已在owner-local合同闭合。`TimelineQuery`新增Turn、Step、Action、Artifact、Effect、Review Case和Checkpoint exact ref；结合既有Identity、Lineage、Instance、Run、Time、Parent、Causation与Correlation，形成完整查询面。

## 语义

- Turn至Checkpoint维度只匹配Event已冻结`ObjectRefs`中的exact ref，不读取或复制其他Owner正文。
- 多个非空维度使用AND语义，不把同一查询扩大成任一匹配。
- 所有新增字段参与canonical Query digest；Cursor签发后任一字段变化均返回`cursor_invalidated`。
- 非空Ref必须是bounded printable ASCII，非法输入在Store读取前失败。
- SQLite无需新增影子事实或第二索引主库；关闭重开后从immutable Event body恢复相同过滤结果。

## 测试证据

- TDD红门：新增测试首先因`TimelineQuery`缺少七个字段编译失败。
- `go test ./contract ./domain ./storage/sqlite`：PASS。
- contract/domain定向ordinary `count=100`与race `count=20`：PASS。
- SQLite reopen定向ordinary `count=100`与race `count=20`：PASS。
- `go test ./...`、`go test -race ./...`、`go vet ./...`：PASS。

## 边界

本切片只增强Continuity查询合同，不把ObjectRef升级为其他领域Fact，不新增跨Owner Reader、写能力、Restore、Provider、CLI root或生产SLA。
