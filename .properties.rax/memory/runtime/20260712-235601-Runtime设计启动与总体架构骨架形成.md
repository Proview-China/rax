# Runtime设计启动与总体架构骨架形成

- 时间：2026-07-12 23:56:01 CST

- 阶段：设计启动；未进入计划

## 事件

用户确认开始搭建Praxis总体架构，并把当前重点固定为Runtime核心执行方法。当前只授权创建设计目录、总体索引和设计草案，不授权创建计划或实现代码。

## 已确认决定

1. Praxis不干涉AI内部认知，Runtime只治理执行生命周期、外部效果和证据；
2. AgentIdentity持久、AgentInstance可替换、AgentRun短暂；
3. Runtime只拥有生命周期和协调权，不吞并其他独立引擎；
4. V1一个AgentInstance独占一个SandboxLease；
5. 身份、正式Memory、Knowledge和确认Asset位于可销毁Sandbox外；
6. 核心执行采用期望状态、状态机和持续Reconcile；
7. 外部观察与干预通过可认证、可授权、可追溯的控制协议完成。
